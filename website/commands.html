Bot Commands

Explore the full capabilities of Discord Bot Universal.

Utility

s?help
Show the help menu.
s?poll <question>
Create a simple yes/no poll.
s?invite
Get the bot's invite link.
s?ping
Check bot latency.
s?serverinfo
View server information.
s?userinfo <user>
View user information.
s?avatar <user>
View a user's avatar.
s?botinfo
View bot statistics.

Moderation

s?ban <user>
Ban a member from the server.
s?kick <user>
Kick a member from the server.
s?timeout <user>
Timeout a member.
s?unban <id>
Unban a user by ID.
s?purge <amount>
Delete a bulk amount of messages.
s?nuke
Delete and recreate the channel.
s?lock
Lock the current channel.
s?unlock
Unlock the current channel.

Fun

s?joke
Get a random joke.
s?fact
Get a random useless fact.
s?meme
Get a random meme.
s?rps
Play Rock, Paper, Scissors.
s?cat
Get a random cat picture.
s?dog
Get a random dog picture.

Leveling

s?rank <user>
View rank card.
s?levels
View the leaderboard.
